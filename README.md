# dev_env

## 安装zsh & oh my zsh

```bash
sudo apt install zsh
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```

在git项目下zsh会有些卡顿

git config --global --add oh-my-zsh.hide-dirty 1

**zsh的主题可以设置为random**

推荐使用ys主题
